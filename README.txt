H2Flow kiosk launcher

Put these two files in the SAME folder as:
- index.html
- styles.css
- app.js
- equipment-data.json
- images/
- videos/ (optional)

Then double-click:
- Start-H2Flow-Kiosk.bat

What it does:
- starts a tiny local web server using built-in PowerShell
- opens Edge in fullscreen kiosk mode
- falls back to Chrome kiosk mode if Edge is not found

To exit kiosk mode:
- Press Alt+F4
